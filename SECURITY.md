# Security Policy

## Supported Versions

This project is currently in active development. Security fixes will be applied to the latest release / main branch.

## Reporting a Vulnerability

If you discover a security issue, please **do not** open a public issue immediately.

Instead, do one of the following:

1. Open a GitHub Security Advisory (preferred), or
2. Email the maintainer(s)

Please include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment (what an attacker could do)
- Any suggested remediation

We will respond as quickly as possible and coordinate a fix and disclosure timeline.
